# 用語集 (Glossary)

ドメインモデリングで確定した用語。コード・ドキュメント・会話でこの名前を使う。

| 用語 | 英名 | 定義 |
|------|------|------|
| アルバム | Album | 今回の成果物。ページの順序付きリスト。1冊のみ(複数プロジェクト管理はしない — ADR-0001) |
| ページ | Page | A4 縦 1枚。1ページ=1テーマ(ADR-0005)。英語タイトルと写真(1〜4枚)を持つ |
| テーマ | Theme | ページの主題(家族・街・趣味など)。Page と 1:1 なので独立エンティティにはしない |
| 写真 | Photo | 1枚の画像とその短い英語キャプション(ADR-0003)。実体は写真フォルダ内のファイル |
| キャプション | Caption | 写真に添える短い英文。1写真につき1つまで |
| レイアウト | Layout | ページ内の写真配置。写真枚数から自動導出される(1枚/2枚/3〜4枚の3型 — ADR-0005)。ユーザーは選ばない |
| コラージュ | Collage | ページ単位のオプション(`Page.layout`)。写真枠を拡大し重なりを許して大きく見せる(ADR-0005 改訂)。既定はグリッド |
| 写真サイズ | Photo.scale | 写真単位の大きさ倍率(0.7〜1.5)。UI では小/標準/大/特大の4段階 |
| 重なり順 | Photo.z | コラージュでの前後関係(奥 -1 / 標準 0 / 手前 +1)。コラージュ時のみ UI に表示 |
| 印刷仕様 | PrintSpec | A4 縦・300dpi・全周 5mm 以上余白・PDF 1ファイル(ADR-0002)。固定値であり設定項目ではない |
| 写真フォルダ | Photo Folder | AirDrop で集めた写真を置く Mac 上の 1 フォルダ(ADR-0006)。アプリの唯一の画像ソース |
| アルバム定義 | album.json | ページ順・写真割り当て・キャプションを保持する唯一の永続状態(ADR-0004) |
| 入稿 | — | 完成 PDF をコンビニのネットワークプリントに登録すること |
