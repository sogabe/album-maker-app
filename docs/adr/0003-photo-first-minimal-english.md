# ADR-0003: 写真中心・文字は短い英語キャプションのみ

- Status: Accepted
- Date: 2026-07-18

## 状況

読者はホストファミリー(英語話者)、制作者は日本語話者。
長文の英訳は制作の時間的ボトルネックになり、息子が会話で説明する余地も減る。

## 決定

アルバムは写真中心とし、文字は **1枚につき短い英語キャプション1つ** に限定する。
本文段落・日本語併記は設けない。

## 帰結

- ドメインモデルのテキストは `Photo.caption`(短い英文)と `Page.title` 程度に収まる。
- 翻訳作業は数十フレーズで済み、制作フローのボトルネックにならない。
- アルバムは「会話のきっかけ」装置になる(指差しで話が広がる)。
