# ADR-0004: ローカル Web アプリとして作る

- Status: Accepted
- Date: 2026-07-18

## 状況

アプリの形態候補は「ローカル Web アプリ」「CLI/スクリプト」「HTML+印刷CSS」。
操作者は父親(必要なら息子も触る)。写真は AirDrop で Mac 上の 1 フォルダに集める
(ADR-0006)。実行環境は macOS + mise(Node LTS 導入済み)。

## 決定

**Mac 上で動かすローカル Web アプリ**とする。ブラウザで写真の割り当て・並べ替え・
キャプション編集を行い、プレビューを見ながら PDF を出力する。

技術構成(最小):

- Node LTS 上の単一プロセス(小さな HTTP サーバ)
- サーバは「写真フォルダの読み取り」「アルバム定義 `album.json` の保存」
  「PDF 生成」だけを担う
- PDF 生成はライブラリで直接行い(例: pdf-lib + sharp で 300dpi へリサイズ)、
  ブラウザの「PDF として保存」には依存しない — ADR-0002 の寸法・余白・
  ファイルサイズ要件をコードで保証するため

## 帰結

- デプロイ・アカウント・DB は無し。状態は `album.json` と写真フォルダのみ。
- ブラウザ差異に印刷品質が左右されない。
- 息子さんもブラウザ画面でキャプションを一緒に書ける。
