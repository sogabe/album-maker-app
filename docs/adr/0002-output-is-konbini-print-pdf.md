# ADR-0002: 出力はコンビニ印刷向け A4 PDF とする

- Status: Accepted
- Date: 2026-07-18

## 状況

印刷手段の選択肢は「自宅プリンタ」「コンビニ印刷」「フォトブック入稿」。
フォトブックは納期(数日〜1週間)が締め切り(2〜3日)と衝突するため除外。
自宅プリンタは画質・用紙の制約がある。

## 決定

成果物は **コンビニのネットワークプリント(セブン netprint / ファミマ・ローソン
ネットワークプリント)に入稿できる A4 PDF** とする。

技術仕様:

- 用紙: A4 縦(210×297mm)
- 解像度: 画像は 300dpi 相当を目標(最低 200dpi)
- 余白: コンビニ複写機はフチなし印刷不可のため、全周 5mm 以上の余白を前提に
  デザインする(塗り足し・トンボは不要)
- ファイル: PDF 1ファイル、サービスのサイズ上限(目安 10〜20MB)以内に収める
- カラー: RGB のままで良い(コンビニ機がハンドリングする)

## 帰結

- ドメインモデルの `PrintSpec` はこの仕様に固定し、可変にしない。
- フチなし(全面写真)デザインは採用しない。
- 画像の再圧縮/リサイズ処理が必要(スマホ写真をそのまま並べるとサイズ超過)。
