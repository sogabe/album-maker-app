# ADR-0001: アプリではなくアルバム完成を最優先する

- Status: Accepted
- Date: 2026-07-18

## 状況

中2の息子が 2026-07 下旬(来週)に米国ホームステイへ出発する。息子の住む場所・趣味などを
紹介する紙のアルバムを持たせたい。印刷済みの現物が **2〜3日以内** に必要。
「アプリで作ってみたい」という希望もあるが、締め切りと衝突する。

## 決定

アルバムの完成を最優先とし、アプリは「今回のアルバムを最短で PDF にするための最小ツール」
に限定する。汎用アルバムメーカー(複数プロジェクト管理、テンプレート市場、アカウント等)は
スコープ外とする。

## 帰結

- 機能追加の判断基準は常に「今回のアルバムが2〜3日以内に刷れるか」。
- 永続化・編集 UI・共有機能などは、アルバム完成に不要なら作らない。
- アプリとして育てたくなったら、アルバム納品後に別途 ADR を起こす。
